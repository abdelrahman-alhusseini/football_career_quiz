How to open this project

1) Unzip the file.
2) Open the folder in VS Code.
3) Open terminal inside the folder.
4) Run:

flutter create .
flutter pub get
flutter run

Important:
This ZIP contains the full Dart/Flutter app source code and pubspec.
The Android/iOS platform folders are generated by `flutter create .` on your computer because Flutter is not installed in this sandbox environment.

If Flutter asks about overwriting files, do NOT overwrite the lib folder or pubspec.yaml. Usually Flutter safely generates missing platform folders.
